# AgroBov Control

Projeto desenvolvido para o Agrinho 2026 com tema ** Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente**.

## Objetivo

O AgroBov Control é um sistema web simples para auxiliar no controle de gado em uma propriedade rural. Ele permite cadastrar animais, controlar raça, peso, idade, vacinação, pastagem, vendas e gerar relatórios básicos.

## Tecnologias utilizadas

- HTML
- CSS
- JavaScript puro

O projeto não utiliza frameworks, bibliotecas externas, CSS inline ou JavaScript inline.

## Funcionalidades

- Cadastro de animais
- Edição e exclusão de registros
- Controle de vacinação
- Controle de pastagem
- Registro de vendas
- Relatório de quantidade de animais
- Relatório de peso médio
- Gráfico simples por raça feito em JavaScript puro
- Cálculo de índice de sustentabilidade
- Layout responsivo

## Como executar

Abra o arquivo `index.html` no navegador.
